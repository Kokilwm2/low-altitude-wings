# 低空之翼 - 低空服务模拟器

这是一个展示低空服务模拟器游戏的官方网站。

## 项目描述

低空之翼是一款创新的低空服务模拟器游戏，玩家可以在游戏中体验：
- 低空配送服务
- 低空预警任务
- 低空救援行动

## 技术栈

- HTML5
- CSS3
- JavaScript
- Font Awesome 图标

## 本地运行

1. 克隆仓库
```bash
git clone https://github.com/Kokilwm2/low-altitude-wings.git
```

2. 打开 index.html 文件即可在浏览器中查看网站
